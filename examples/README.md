# SE3040 Lab 03 - Examples

Run any example from the repository root:

```bash
node examples/read-file.js
node examples/write-file.js
node examples/web-server.js
node examples/http-request.js
node examples/use-module.js
node examples/promises.js
node examples/async-await.js
```

Or via npm scripts:

```bash
npm run read-file
npm run write-file
npm run web-server
npm run http-request
npm run use-module
npm run promises
npm run async-await
```

Notes:
- `examples/web-server.js` listens on `http://localhost:8080` (stop with `Ctrl+C`).
- `examples/write-file.js` overwrites `examples/file.txt`.
